# Play Scala based application for [backlog](https://en.wikipedia.org/wiki/Backlog) maintenance

This is application for [backlog](https://en.wikipedia.org/wiki/Backlog) maintenance and estimation.\
The application now is under construction.

## Running

Run this using [sbt](http://www.scala-sbt.org/).

```
sbt run
```

And then go to http://localhost:9000 to see the running web application.

## Heroku

Application also available on Heroku servers. Try [rph](https://rph.herokuapp.com/)
